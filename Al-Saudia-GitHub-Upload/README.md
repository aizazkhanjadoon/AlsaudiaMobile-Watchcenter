# Al-Saudia Mobile & Watch Centre

Static website for Al-Saudia Mobile & Watch Centre.

## Files
- index.html: Home page
- shop.html: Full shop/catalog page
- styles.css: Site styling
- products.js: Product catalog data
- app.js: Filtering, search, and selection drawer
- images/: All product and store images

## GitHub Pages
Upload this folder contents to a GitHub repository. Set GitHub Pages to deploy from the main branch root, then open index.html as the home page.
